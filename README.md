# Мастер пол

Система для автоматизации управления данными партнёров компании, занимающейся производством и реализацией продукции. Позволяет добавлять, редактировать и просматривать информацию о партнёрах, отслеживать историю продаж продукции через них, а также автоматически рассчитывать индивидуальные скидки на основе объёма реализации.

## Начало работы

Эти инструкции предоставят вам копию проекта и помогут запустить на вашем локальном компьютере для разработки и тестирования.

### Необходимые условия

Для работы системы необходимо установить:
Python 3.8 или выше
pip — менеджер пакетов Python
django - для создания самого сайта
psycopg2-binary - для миграций
PostgreSQL — СУБД для хранения данных

### Установка
1. Скачивание и разархивация проекта

2. Открытие проекта через Pycharm

3. Установка дополнительных компоентов
pip inastall django
pip install psycopg2-binary

4. Включение вашей БД паралельно с проектом

5. Перейдите в файл settings.py и укажите ваши параметры подключения к БД
DATABASES = {
    'default': {
        'ENGINE': 'django.db.backends.postgresql',
        'NAME': 'masterpol_db',
        'USER': 'django_user',
        'PASSWORD': 'your_password',
        'HOST': 'localhost',
        'PORT': '5432',
    }
}

6. Создайте и примените структуру БД

python manage.py makemigrations
python manage.py migrate

7. для запуска программы введите 

python manage.py runserver

8. откройте веб-сервер по ссылке
 
http://127.0.0.1:8000

9. проверяйте работу, переходя по вкладкам и изучая веб-сайт
